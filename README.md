# BSHOS
Baltic Sloboda Home OS - Система для контроля отопления в загородном доме по СМС
## Платформа:
- Arduino leonardo
- GPRS Shield от Амперки
- Аналоговый датчик температуры
- Релейный модуль на 2 канала
